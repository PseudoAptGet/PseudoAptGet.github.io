A Brief Overview on the Purpose of this Website.

The purpose of this website (Yes, I know it sounds like some college student writing a lab report or thesis) is to have fun with bootstrap by building a website, and focus the content of the website on web-accessibility requirements. I will go over the accessibility criteria under WCAG 2.1, and try to implement such criteria while experimenting with bootstrap.

Naturally, that means that I will have to implement some of the bootstrap solutions to accessibility requirements, and this website will document what the solutions I know of are, and briefly go over how to implement them. For further information, the code for this website is hosted on github, and the github link to my profile page is in the "contact me" section of the navbar. You can, if you're feeling adventerous, also simply use inspect element, and see the code yourself in the localized context.

In the spirit of making this website accessible, and meeting the operable/ navigatable section of WCAG 2.1, I recommend that you see the navbar's 4 tenets section, and check each of them out.
