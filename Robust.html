Robust

There really isn't much to say on this recommendation (and in my defense, this was the shortest section in WCAG, as well as the one with the vaguest recommendations. Two major points are made.

  1. All starting tags must have ending tags so that assistive technology like screenreaders can work properly (and so that I can scrape the website's data without creating special exceptions in my python script to handle poor coding habits).
  2. User interface components should have unique id's and the role can be easily determined.
